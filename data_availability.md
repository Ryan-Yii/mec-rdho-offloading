# Data and Code Availability

This repository contains the code and generated experimental artefacts used for
the RDHO mobile edge computing experiments.

## Data Source

The experiments use simulated MEC task sets generated by the scripts in this
repository. No proprietary, confidential, or human-subject data are used.

Task parameters and task-generation ranges are available in:

- `results/raw/task_parameters.csv`
- `results/raw/task_generation_ranges.csv`
- `paper_tables/task_generation_ranges.md`

## Code Availability

The source code is organised as follows:

- `src/`: core system model, metrics, task generation, and algorithm
  implementations.
- `experiments/`: reproducible experiment entry points and result analysis.
- `configs/`: YAML configuration files for the main, ablation, scalability, and
  sensitivity experiments.
- `tests/`: validation tests for task generation, metrics, and the experiment
  pipeline.

The public repository is available at:

https://github.com/Ryan-Yii/mec-rdho-offloading

## Result Artefacts

Raw outputs, summary statistics, statistical-test results, and generated
figures are included under:

- `results/raw/`
- `results/summary/`
- `results/figures/`
- `results/sensitivity/`
- `paper_tables/`
- `figures/`

The top-level `figures/` directory contains manuscript-facing copies of the
paper figures. The source result figures remain under `results/figures/` and
`results/sensitivity/figures/`.

## Reproducibility Notes

The main benchmark uses 20 mobile devices, 4 edge servers, 2 cloud servers, 40
heterogeneous tasks, a population size of 50, 150 maximum iterations, and 30
independent runs. For a fixed seed, all compared algorithms use the same task
set and network configuration.

To reproduce the validation checks and experiments, install the dependencies in
`requirements.txt` and run the commands listed in `README.md`.
